This is all my work for the Kaggle Housing competition - "House Prices - Advanced Regression Techniques" where I used python as my choice of coding language in order to predict housing prices. The project focused on predicting housing prices using a log-linear model, chosen based on visual analysis that indicated a logarithmic relationship between housing features and prices. The dataset included attributes such as dwelling type, zoning, lot size, street and alley type, utilities, and various property characteristics. The methodology involved categorizing these attributes into groups like garages, location, and condition, and then analyzing their significance. The model was developed with the formula: np.log(SalePrice) ~ GarageCars + OverallQual/Age + BsmtFinSF1 + BsmtQual + FirFlrSF*GrLivArea + KitchenAbvGr*KitchenQual + CentralAir + Fireplaces, incorporating both main effects and interaction terms.
